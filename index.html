웹 미술의 진화 🌐

HTML, CSS, 자바스크립트가 예술을 만났을 때

본 사이트는 일러스트레이터를 희망하는 조은성이 웹 기술과 디지털 아트의 상호작용을 연구하고 직접 그린 결과물입니다. 웹 미술의 발전 과정은 기술 변화가 대중문화의 생산 및 소비 구조를 혁신한 사례입니다. 초기 픽셀 아트부터 현재의 NFT까지, 웹은 예술의 접근성을 극대화하고 소비 형태를 쌍방향적 참여로 확장하며 일러스트레이터를 대중문화 콘텐츠의 핵심 생산자로 격상시켰습니다. 이러한 연구를 통해 기술 변화와 대중의 요구를 선도할 일러스트레이터의 역할에 대한 통찰력을 갖추었습니다. 웹이 예술의 캔버스가 되어가는 과정을 함께 탐험해 보세요.

탐험 시작하기 ↓

🕰️ 1세대: 웹의 탄생과 픽셀 아트 (1990s)

키워드: GIF, 픽셀 아트, 웹 배너

웹이 처음 대중화되던 시기, 느린 인터넷 속도와 제한된 컬러 팔레트로 인해 픽셀 아트(Pixel Art)와 애니메이션 GIF가 웹 미술의 주류였습니다. 단순하지만 생동감 있는 디자인이 유행했으며, 상업적으로는 원시적인 웹 배너가 등장했습니다.

귀여운 키링 컨셉 픽셀 아트 예시

🚀 2세대: Flash와 인터랙티브 아트 (2000s)

키워드: 플래시(Flash), 인터랙션, 웹 애니메이션

Adobe Flash(당시 Macromedia Flash)의 등장으로 웹 미술은 폭발적으로 성장했습니다. 사용자와 상호작용하는 인터랙티브 웹사이트, 복잡한 웹 애니메이션, 그리고 웹툰의 전신이라 불리는 플래시 애니메이션이 웹 미술의 영역을 넓혔습니다. 이 시기는 화려함과 실험성이 강조되었습니다.

00년대에 유행했던 플래시 애니메이션 스타일

🎨 3세대: 반응형 디자인과 웹 GL (현재)

키워드: 반응형, 웹 GL, 제너레이티브 아트

HTML5, CSS3, 자바스크립트의 발전은 Flash를 대체했습니다. 이제 웹 미술은 모든 디바이스에 대응하는 반응형 디자인을 기본으로 합니다. WebGL을 이용한 3D 그래픽, 더욱 발전하는 그림 소프트웨어, 코드가 스스로 예술을 만들어내는 제너레이티브 아트(AI 그림)등 기술과 예술의 경계가 무너지는 최첨단 웹 미술이 발전하고 있습니다.
